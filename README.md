### Installation/Update

	- git clone https://github.com/satyakami/grub2-minimalistic.git
	- cd grub2-minimalistic
	- chmod 755 install.sh
	- sudo ./install.sh
	- cd ..
	- rm -r grub2-minimalistic

### Done!!!
